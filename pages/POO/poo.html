Funciones constructoras

Las funciones constructoras son funciones en JavaScript que se utilizan para crear y inicializar objetos. Se suelen utilizar en conjunción con el operador new para crear nuevas instancias de un tipo de objeto específico. Cuando una función se utiliza como constructor (es decir, se llama con new), se creará un nuevo objeto y this dentro de la función hará referencia a ese nuevo objeto.
