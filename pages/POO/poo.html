Funciones constructoras

Las funciones constructoras son funciones en JavaScript que se utilizan para crear y inicializar objetos. Se suelen utilizar en conjunción con el operador new para crear nuevas instancias de un tipo de objeto específico. Cuando una función se utiliza como constructor (es decir, se llama con new), se creará un nuevo objeto y this dentro de la función hará referencia a ese nuevo objeto.


¿Qué es una clase?

Las clases son una forma de organizar y estructurar tu código al definir cómo deberían ser tus objetos y qué pueden hacer. Permiten crear múltiples objetos con las mismas propiedades y métodos de manera más eficiente y organizada. 

                        class Persona {
                          constructor(nombre, edad) {
                            this.nombre = nombre;
                            this.edad = edad;
                          }
                        }
                        const persona1 = new Persona('Juan', 20);
                        console.log(persona1.nombre); // Juan
                        console.log(persona1.edad); // 20

Las clases se pueden extender

Supongo que creamos una clase general de usuarios, a su vez esos usuarios se ramifican en estudiantes, profesores, empleados ,etc. Claramente tendrán keys en común (propiedades) como ser; nombre, apellido, email, password. Entonces definimos nuestra clase “padre”

                        class Persona {
                          constructor(nombre, edad) {
                            this.nombre = nombre;
                            this.edad = edad;
                          }
                        }
                        class Estudiante extends Persona {
                          constructor(nombre, edad, curso) {
                            super(nombre, edad);
                            this.curso = curso;
                          }
                        }
                        const estudiante1 = new Estudiante('Juan', 20, 'JavaScript');
                        console.log(estudiante1.nombre); // Juan
                        console.log(estudiante1.edad); // 20
                        console.log(estudiante1.curso); // JavaScript
